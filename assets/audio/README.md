# Música de Fondo

Para añadir música de fondo al sitio web, coloca tu archivo de audio en esta carpeta con el nombre `fondo.mp3` o `fondo.ogg`.

## Formatos soportados:
- MP3 (recomendado)
- OGG (alternativo)

## Recomendaciones:
- Duración: 2-5 minutos (se repetirá automáticamente)
- Género: Indie, rock alternativo o música suave
- Calidad: 128-192 kbps
- Tamaño: Máximo 10MB

## Archivo esperado:
- `fondo.mp3` - Archivo principal de audio
- `fondo.ogg` - Versión alternativa (opcional)

Una vez que añadas el archivo, la música comenzará automáticamente cuando el usuario haga clic en la entrada.
